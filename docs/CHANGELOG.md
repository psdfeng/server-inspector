# Changelog

## 2026-03-02

- 新增 macOS 巡检支持
- 优化 macOS 磁盘统计，修复伪文件系统导致的 100% 误报
- 巡检记录支持删除
- 报告导出支持按指定巡检记录导出
- 仪表盘“未处理告警/最近巡检”默认显示 3 条
- 新增每日定时自动巡检设置
- 优化菜单和仪表盘图标显示
- Windows 巡检改为 WinRM 方案，适配无 SSH 的常见 Windows 主机
- Windows 巡检增强自动探测（5985/5986，多协议回退）
- 修复 Windows CPU 异常值显示（超大百分比）
- 新增输入校验（IP、端口、重复 IP 等）
- 新增 `/healthz` 健康检查接口
- 增加关键查询索引，提升列表和告警查询性能
